# Superstore Sales Data Analysis

## 📌 Description
Performed end-to-end data analysis on a retail dataset using Python and Power BI to extract meaningful business insights.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- Power BI

## 🔍 Key Insights
- West region recorded highest sales
- Technology category generated maximum profit
- Top 10 products identified based on sales
- Sales trend increased over time

## 📊 Dashboard
Created an interactive Power BI dashboard for visualization of sales and profit data
